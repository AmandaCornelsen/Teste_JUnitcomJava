# JUnit Calculator – Digital Calculator with Unit Testing

This project showcases a **digital calculator implemented in Java**, built as a practical exercise to deepen understanding of **JUnit testing** and **best practices in software testing**.

---

##  Tech Stack
- **Java (latest version)** – core language used.
- **JUnit** – framework for writing and running unit tests.
- **Clean Code & Testing Practices** – focusing on modularity, clarity, and maintainability.

---

##  Project Structure
Teste_JUnitcomJava/

│ ├── src/main/java/... (Calculator class and related logic)
│ ├── src/test/java/... (JUnit test classes)
├── README.md

##  Features & Testing Highlights
- **Basic arithmetic operations**: addition, subtraction, multiplication, division.
- **Edge case handling**: divide-by-zero checks, invalid input handling, etc.
- **High test coverage** using **JUnit**, demonstrating rigour and attention to detail.
- **Test-driven development mindset**, with clear naming and organization of test cases.
